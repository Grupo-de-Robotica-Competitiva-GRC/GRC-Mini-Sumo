# GRC-Mini-Sumo

## Descrição

O presente projeto se trata de um robô de competições de sumo, da modalidade de mini sumo autônomo. O robô utiliza sensores para detecção de oponentes e se dispoê de uma lógica de combate para empurrar o adversário para fora da arena, tudo isso enquanto evita, por si próprio, sair da arena. O projeto engloba eletrônica, embarcados, desenvolvimento de firmware, modelagem de controle e modelagem 3D.
